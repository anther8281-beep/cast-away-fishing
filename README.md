# Cast Away 🎣

A self-contained browser fishing game. Charge your cast for distance, set the hook
the instant a fish bites, then keep it inside the bar to reel it in. Sell your catch,
upgrade your gear, and chase the legends of the deep.

## Play
Open `index.html` in any modern browser — or visit the live link.

## Notes
- Single HTML file. HTML5 Canvas + WebAudio, no build step, no dependencies.
- **No network calls, no external scripts, no analytics, no tracking.**
- Progress saves locally in your browser (`localStorage`) only.
